# C-snake-game-using-SFML
a university project of a snake game made by C++ and the SFML (Simple and Fast Multimedia Library)
# About
a simple snake game where you play as a snake and try to eat the apples that spawn randomly, the score is written and recorded on top left of the screen (the current score and the highest score achieved), you only lose if you hit yourself meaning you can pass through the wall and come from the opposite one.
I developped this with the contribution of [@Robot404a](https://github.com/Robot404a) who helped with the most parts of this project.

## Requirements
This project uses [SFML 3.0.0](https://www.sfml-dev.org/) (Simple and Fast Multimedia Library) for graphics, audio, and window handling.
